# Proyecto NestJS para aprendizaje

## Descripción

Es un proyecto orientado al aprendizaje de este gran framework nestjs para nodejs.

## Instalación

```bash
$ npm install
```

## Ejecutando la app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
